# 💳 Modern Digital Banking Dashboard

## 📌 Project Overview

Modern Digital Banking Dashboard is a full-stack financial management application designed to unify multiple banking functionalities into a single platform. It enables users to manage accounts, track transactions, monitor budgets, handle bills, and gain actionable financial insights through an interactive dashboard.

---

## 🚀 Key Features

* Multi-account overview with real-time transaction feeds
* Expense categorization and monthly budget tracking
* Bill management with reminders and workflows
* Rewards tracking and insights
* Currency conversion summaries
* Alerts for low balance, bill due, and budget limits

---

## 🏗️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** FastAPI
* **Database:** PostgreSQL
* **Authentication:** JWT (Access & Refresh Tokens)
* **Integrations:** ExchangeRate API, Open Banking APIs (Plaid/Open Bank Project)
* **Extras:** Webhooks / Polling for account updates

---

## 🧩 Modules

* **Module A:** Authentication & User Setup
* **Module B:** Accounts & Transactions
* **Module C:** Budgeting & Categorization
* **Module D:** Bills, Reminders & Rewards
* **Module E:** Insights, Alerts & Admin

---

## 📊 System Architecture

The application follows a modular full-stack architecture:

* React frontend communicates with FastAPI backend via REST APIs
* Backend handles authentication, business logic, and API integrations
* PostgreSQL stores structured financial data
* Background services handle reminders and alerts

---

## 🗂️ Database Schema (Core Entities)

* **Users** – User details and authentication
* **Accounts** – Bank accounts and balances
* **Transactions** – Financial transaction records
* **Budgets** – Monthly category-based budgets
* **Bills** – Payment tracking and reminders
* **Rewards** – Loyalty and points tracking
* **Alerts** – System-generated notifications
* **AdminLogs** – Administrative actions tracking
